# 🚀 Sleek AI - Build Fast, Build Sleek

A high-performance, modern AI-powered platform designed for rapid product generation and seamless project management. **Sleek AI** combines the power of Next.js with the versatility of the **InsForge** backend to deliver a top-tier user experience.

---

## ✨ Key Features

- 🤖 **AI-Driven Chat Interface**: An intuitive workspace to interact with AI agents for brainstorming and project generation.
- 📐 **Dynamic Canvas**: A visual environment to manage and organize your project's pages and components.
- ⚡ **Next.js & React 19**: Built on the absolute bleeding edge of the React ecosystem.
- 🎨 **Minimalist Design System**: Leveraging Tailwind CSS 4+ and Shadcn UI for a truly "sleek" and modern look.
- 🔌 **InsForge Integration**: Fully integrated with the InsForge BaaS for database, authentication, and serverless functions.
- 🌑 **Dark Mode by Default**: Optimized for productivity and focus.

---

## 🛠️ Tech Stack

### Core
- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Library**: [React 19](https://react.dev/)
- **Backend / BaaS**: [InsForge SDK](https://insforge.app/)
- **AI Management**: [Vercel AI SDK](https://sdk.vercel.ai/)

### Styling & UI
- **CSS Framework**: [Tailwind CSS 4+](https://tailwindcss.com/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/) & [Phosphor Icons](https://phosphoricons.com/)

### State Management
- **Hooks**: Custom `useCanvas` & `nuqs` for query-based state persistence.
- **Data Fetching**: [TanStack Query v5](https://tanstack.com/query/latest)
